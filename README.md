# CubeZone

## Project Idea and Problem Statement

### Idea
>The idea for the project is to create an e-commerce website for speed cubes.

### Problem Statement
>Since many top brands from the speed cube industry do not have a physical retail platform to sell their products in my region, this project aims to address this gap by designing an e-commerce website with direct dealership with these top brands, thus making their products accessible to the local community.
